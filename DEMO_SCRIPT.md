## Recording Tips

### Pacing:
- Speak clearly and confidently
- Pause for 1-2 seconds between sections
- Let animations complete (don't rush)
- Allow 3-5 seconds for viewers to see results

### Visual Polish:
- Smooth mouse movements (not too fast)
- Highlight cursor clicks (use recording software feature)
- Keep browser chrome visible (shows it's real Chrome)
- Show loading states (sped up 2-3x in editing)

### Narration:
- Record in quiet environment
- Use quality microphone
- Enthusiastic but professional tone
- Emphasize key words: "ALL FOUR APIs", "ANY webpage", "100% local"
- Smile while narrating (it shows in your voice!)

### Editing:
- Add smooth transitions between sections
- Speed up loading animations (2-3x)
- Add subtle background music (royalty-free, low volume)
- Add text overlays at key moments
- Color grade for consistency
- Add subtle zoom-ins on important UI elements
