## Known Limitations (Documented)

1. **Chrome AI APIs Required**
   - Only works on Chrome Dev/Canary (v128+)
   - Requires API flags enabled
   - Not for production Chrome yet

2. **Model Download Required**
   - Gemini Nano: ~1.7GB download
   - One-time setup

3. **Processing Time**
   - AI transformations: 5-15 seconds
   - Depends on content length

4. **Internet Required**
   - Mermaid CDN for diagram rendering
   - Could be made offline with local Mermaid.js

5. **Storage Limit**
   - 100 transformations max (configurable)
   - Can be increased in settings
